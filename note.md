## npm vs yarn

Yarn是新一代包管理工具，修复了npm的一些缺陷。
它速度快、安装版本统一、更安全、更简洁的输出、更好的语义化。
yarn init初始化项目。
yarn add xxx 添加包
yarn remove xxx 删除包
yarn/yarn install 安装项目里所有依赖包
